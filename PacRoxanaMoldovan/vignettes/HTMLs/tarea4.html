6. An√°lisis de grupos o conjuntos de genes.

En este apartado estudiaremos si existe relaci√≥n entre conjuntos de genes previamente definidos, en este caso tomaremos las colecciones de genes de Go y KEGG previamente creadas, y el fenotipo.

En este tipo de an√°lisis se formulan las siguiente hip√≥tesis nulas:

  • Q1 : Los genes de un conjunto muestran el mismo patr√≥n de asociaci√≥n con el fenotipo que los genes pertenecientes al resto de grupos. Esta hip√≥tesis se centra en la comparaci√≥n entre la asociaci√≥n de los genes pertenecientes a un conjunto a un fenotipo y la asociaci√≥n de los genes de otros conjuntos y ese fenotipo. Para esto haremos un test competitivo.

  • Q2 : El conjunto de genes no presenta ning√∫n gen con un nivel de expresi√≥n asociado al fenotipo estudiado. Esta hip√≥tesis estudia la expresi√≥n diferencial de los genes pertenecientes a cada conjunto. Para esto haremos un test autocontenido.

GSA para grupos KEGG:

En este apartado emplearemos EnrichmentBrowser para llevar a cabo un an√°lisis GSA. Mediante este test podemos encontrar los grupos cuyos genes muestran diferencias estad√≠sticamente significativas entre dos estados o fenotipos, en este caso el tratamiento con formaldeh√≠do.
Este an√°lisis difiere del GSEA en que nos da una medida del enriquecimiento (estad√≠sitico maxnean), este toma la media de las partes positivas (grupos de genes positivos, es decir que en la segunda condici√≥n han aumentando su expresi√≥n) y la media de las partes negativas (grupos de genes negativos, es decir que en la segunda condici√≥n han disminuido su expresi√≥n) y se queda con el m√°ximo de ambos.
Para el ajuste de los p valores usamos el m√©todo de Benjamini-Hochberg.

Gene set analysis con tami para grupos Go

En este apartado emplearemos el paquete tami con el fin de llevar a cabo un an√°lisis de grupos de genes tanto competitivo como autocontenido usando la funci√≥n GeneSetTest con los siguientes par√°metros:

  • test -> rowtest
    Realizar un an√°lisis de expresi√≥n diferencial marginal (test de la t moderada de Limma).
  • association -> ‚Äústatistic‚Äù
    Empleamos el estad√≠stico de contraste como asociaci√≥n entre la expresi√≥n y el fenotipo.
  • correction -> ‚ÄúBH‚Äù
    Como m√©todo de ajuste de los p-valores el m√©todo de Benjamini y Hochberg.
  • GeneNullDistr -> ‚Äúrandomization‚Äù. Contrastamos la distribuci√≥n nula de aleatorizaci√≥n por columna.
  • alternative -> ‚Äútwo-sided‚Äù. Usamos un test bilateral.
  • nmax -> ‚Äú100‚Äù. N√∫mero de remuestreos aleatorios.

Hip√≥tesis nula competitiva

La hip√≥tesis nula Ho indica que los genes de un grupo previamente definido S est√°n al menos tan frecuentemente diferentemente expresados como los genes en el grupo Sc.
Par√°metros:

  • GeneSetNullDistr = ‚Äúcompetitive‚Äù
geod75759_GO_comp <- GeneSetTest(x = geod75759_rma,
                              y=factor(pData(geod75759_rma)$Sample_treatment),
                              test = rowtmod, association="statistic",
                              correction="BH", GeneNullDistr = "randomization",
                              GeneSetNullDistr = "competitive",
                              alternative="two-sided", nmax = 100,
                              id = "ENTREZID", gsc=geneIds(mccGOgsc),

geod75759_GO_comp_df = tidy(geod75759_GO_comp)
